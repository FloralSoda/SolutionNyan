No Javascript

Hey scriptless friend! This is the No-JS version of the site. This operates by running a precomputed snapshot of the js version, so that all rendering has been done for you :) It's updated less often for now, as I manually render it out on my PC, but when this website has its own backend the server will automatically render it all for you and this version of the site will likely merge with the main site where the JS is just simply optional.

If you're worried about cookies, the base site does not interact with cookies unless you explicitly interact with a customisation feature (like changing the theme). If you're worried about tracking, none of my code does any form of tracking whatsoever, so the only tracking would be coming from Github if there even is any. This will forever remain the case.
